# 🚀 DÉMARRAGE RAPIDE - Vault Key

## 5 minutes pour lancer votre e-commerce!

### 1️⃣ Installation locale (2 min)

```bash
npm install
npm run dev
```

Ouvrir: http://localhost:5173

### 2️⃣ Accès Admin (1 min)

```
URL: http://localhost:5173/admin
Mot de passe: vault_admin_2024
```

Ajouter/modifier/supprimer des produits!

### 3️⃣ Tester l'achat (1 min)

1. Ajouter un produit au panier
2. Aller au checkout
3. PayPal est prêt à tester

### 4️⃣ Déployer sur Vercel (1 min)

```bash
# Push sur GitHub
git add .
git commit -m "Initial"
git push

# Puis:
# 1. Aller sur vercel.com
# 2. Importer le repo GitHub
# 3. "Deploy"
# 4. C'est live! 🎉
```

## 📋 Checklist avant production

## 🎯 Prochaines étapes

1. **Personnaliser les couleurs**
   - Modifier tailwind.config.js

2. **Ajouter des produits**
   - Admin → Ajouter produit

3. **Configurer le domaine**
   - Vercel → Settings → Domains

4. **Ajouter Google Analytics**
   - Ajouter dans index.html

5. **Implémenter une BDD**
   - MongoDB/PostgreSQL
   - API Node.js/Express

## 💡 Tips

- Vos données sont sauvegardées dans localStorage
- F12 pour debug dans la console
- `localStorage.clear()` pour reset
