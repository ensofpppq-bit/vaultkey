# Guide Utilisateur - Vault Key

Bienvenue sur **Vault Key**, votre boutique en ligne de clés USB premium!

## 🛍️ Comment acheter

### 1️⃣ Parcourir les produits
- Aller sur la page d'accueil
- Voir tous les produits disponibles
- Vérifier le stock et le prix

### 2️⃣ Ajouter au panier
- Cliquer sur **"Ajouter"** sur le produit
- La quantité s'ajoute au panier
- Voir le panier en haut à droite

### 3️⃣ Gérer le panier
- Cliquer sur l'icône **panier** en haut
- Voir tous les articles
- **Modifier la quantité**: +/- boutons
- **Supprimer un article**: cliquer sur la corbeille
- Voir le total avec frais de port

### 4️⃣ Passer la commande
- Cliquer sur **"Passer la commande"**
- Vérifier le résumé
- Cliquer sur le bouton **PayPal**
- Se connecter à PayPal
- Approuver et payer

### 5️⃣ Confirmation
- Voir le message "Commande confirmée"
- Reçevoir un email de confirmation
- Commande sera expédiée rapidement
